# 선체 설계 시수 관리 시스탬

<aside>
💡 선체 설계 시수 관리 시스탬 입니다.

</aside>

**install**

git clone 후에

```tsx
yarn;
```

**start**

프로그램 실행

```tsx
yarn electron:serve
```

서버 실행

```tsx
cd server && yarn start
```

---

### 개발환경

```tsx
Front : React.js, Electron, css module, antd, yarn, context API, sheetjs

Back :  Node, mysql
```
